# JMComic 漫画下载器 for AstrBot

基于 [JMComic-Crawler-Python](https://github.com/hect0x7/JMComic-Crawler-Python) API 的 AstrBot 插件，支持在 QQ 群聊和私聊中下载禁漫天堂漫画并转为 PDF 上传。

## 安装依赖

在 AstrBot 的 Python 环境中安装：

```bash
pip install jmcomic Pillow
```

> ⚠️ 注意：必须在 **AstrBot 所使用的 Python 环境** 中安装，不是系统 Python。
> 如果不确定，可以在 AstrBot WebUI → 插件管理 → 打开终端，执行 `pip install jmcomic Pillow`。

## 使用命令

### 漫画搜索与下载（所有人可用）

| 命令 | 说明 |
|------|------|
| `/jm search <关键词> [页码]` | 智能搜索漫画（支持多关键词如 `原神 刻晴`）。搜索结果自带违禁过滤，并自动附带**每一本漫画的高清封面预览图**！ |
| `/jm page <页码>` | 智能连贯翻页！系统会自动记忆你的上一条搜索记录，直接发送 `/jm page 2` 即可无缝翻页。 |
| `/jm <编号>` | 下载指定编号的漫画 → 转PDF → 上传群文件。已下载的直接发送缓存文件。 |
| `/jm batch <编号> [编号]...` | **批量下载**！支持同时输入多个车号，逗号或空格隔开均可（如 `123, 456 789，111`），后台全自动排队下载。 |
| `/jm list` | 列出当前服务器上已缓存的漫画列表。 |

### 批量下载管理（仅管理员）

| 命令 | 说明 |
|------|------|
| `/jm batch off` / `on` | 关闭 / 开启 批量下载功能，防止群友滥用占用服务器资源。 |
| `/jm batch max <数量>` | 限制每次批量下载的**最大本数**（默认为 10 本，设为 `-1` 则无上限）。 |

### 漫画及常规管理（仅管理员）

| 命令 | 说明 |
|------|------|
| `/jm delete <编号>` | 删除服务器上的指定漫画缓存 |
| `/jm delete all` | 删除全部漫画缓存（需二次确认） |

### 群聊与权限控制（仅管理员）

| 命令 | 说明 |
|------|------|
| `/jm group off <群号>` / `on` | 对该群禁用 / 恢复 JM 下载功能 |
| `/jm black add <QQ号>` / `remove` | 拉黑用户，禁止其使用本插件 / 移除拉黑 |
| `/jm black list` | 查看被拉黑的用户名单 |

### 黑名单自动风控体系（仅管理员）

_包含单本黑名单和基于标签的自动拦截风控。作用域支持：`global`(全局)、`group <群号>`、`user <QQ号>`_

| 命令 | 说明 |
|------|------|
| `/jm black_jm add <编号>` | 拉黑指定漫画，禁止任何人下载它 |
| `/jm black_jm remove <编号>` | 移除漫画拉黑限制 |
| `/jm black_tag add <作用域> <标签>` | 拉黑特定标签。当用户搜索或下载包含该标签的漫画时，系统会在后台**自动触发拦截**！ |
| `/jm black_tag remove <作用域> <标签>` | 移除指定作用域的标签拉黑 |
| `/jm black_tag list <作用域>` | 查看该作用域已拉黑的所有标签 |
| `/jm black_tag remove_all <作用域> confirm` | **一键清空**该作用域的所有标签黑名单限制 |

## 核心特性

- **视觉进化**：搜索结果全自动携带高清封面排版，所见即所得。
- **并发批量下载**：支持多任务智能排队，自动提取一切合法车号，下载不再苦等。
- **5分钟缓存释放**：后台智能打包 ZIP，5 分钟内无人下载自动删除庞大的原始 PDF，极致节省硬盘空间。
- **多维度风控防御体系**：群聊封禁、单人拉黑、单本拉黑、标签自动拦截，彻底杜绝违规风险。
- **兼容自定义前缀**：全面适配任意 AstrBot 唤醒前缀（`/` `#` `>>` 等）。
